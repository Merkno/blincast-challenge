# Blincast Challenge

Sistema simplificado com servidor HTTP e cliente em Python.

## Tecnologias

- Python 3
- Flask
- Requests
- HTML5

## Estrutura

- `server.py` -> Servidor HTTP
- `client.py` -> Cliente interativo via terminal
- `templates/index.html` -> Cliente web em HTML
- `database.json` -> Armazenamento local
- `requirements.txt` -> Dependências

## Como rodar

```
- git clone https://github.com/Merkno/blincast-challenge
- cd blincast-challenge
- pip install -r requirements.txt
- python server.py
```
