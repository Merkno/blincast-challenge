# Blincast Challenge

Sistema simplificado com servidor HTTP e cliente em Python.

## Tecnologias

- Python 3
- Flask
- Requests

## Estrutura

- `server.py` -> Servidor HTTP
- `client.py` -> Cliente interativo
- `database.json` -> Armazenamento local
- `requirements.txt` -> Dependências
